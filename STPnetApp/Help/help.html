Создание объектов

Мост (Коммутатор) создается нажатием ПКМ в пустом месте программы. В появившемся окне назначается идентификатор и приоритет моста.
У моста отображается его идентификатор и в скобках приоритет

Порт создается нажатием ПКМ по мосту и выбором соответствующего пункта из всплывающего меню. В появившемся окне назначается номер порта.
У порта в левом верхнем углу отображается номер, а в середине - текущее значение, сохраненное в его памяти

Связь создается путем поочередного нажатия на порты разных мостов. Порядок действий: нажмите ЛКМ на один из портов. Информация о нем должна появиться внизу экрана, затем нажмите ЛКМ на другой порт другого моста. Появится окно, где необходимо задать вес для новой связи.
Рядом со связью отображается ее вес


Изменение параметров объектов

Приоритет моста и вес связи можно изменить нажатием по объекту ПКМ и выборе из всплывающего меню пункта редактировать мост/связь. В появившемся окне вводятся новые данные. Параметры порта изменить нельзя


Удаление объектов

Любой объект удаляется нажатием по нему ПКМ и выборе из всплывающего меню пункта удалить мост/порт/связь. При удалении связи, представленной в виде Hub, отдельно удаляются все связи центра с портами, пока Hub не превратится в обычную связь


Перемещение и масштабирование

Нажав и зажав ЛКМ в пустом месте рабочей области программы, можно перемещать все элементы путем перемещения мышки

Нажав и зажав ЛКМ на коммутаторе или порту, можно премещать эти элементы аналогичным способом

Связь можно перемещать только в режиме Hub, когда имеется более двух связей. Для этого необходимо перемещать точку соединения связей

Масштабирование поля программы осуществляется с помощью колесика мышки


Моделирование

Для начала моделирования во вкладке "Этапы" необходимо выбрать один из пунктов.
Рекомендуется использовать горячие клавиши:
Нахождение корневого моста [Ctrl + 1]
Нахождение корневых портов [Ctrl + 2]
Нахождение назначенных и заблокированных портов [Ctrl + 3]

Условные обозначения результатов моделирования:
Корневой мост - зеленый
корневой порт - зеленый
Назначенный порт - синий
Заблокиованный порт - красный


Моделирование по шагам

Для включения данного режима необходимо во вкладке "Моделирование" нажать на пункт "По шагам", рядом с этой надписью появится зеленая галочка

Моделирование запускается аналогично тому, как это делалось в обычно режиме

Теперь для очередного шага необхедимо во вкладке "Моделирование" нажимать на пункт "Следующий шаг". Рекомендуется использовать горячие клавиши [Ctrl + Q]

Завершить моделирование досрочно можно нажатием во вкладке "Моделирование" пункта "Завершить". Рекомендуется использовать горячие клавиши [Ctrl + E]

В данном режиме путь пакета указывается стрелками. Текущий порт, в котором находится пакет, отображается желтым. Другие порты текущего моста отображаются фиолетовым, если через этот порт пакет уже выходил далее. Когда все порты, кроме текущего, будут отмечены как пройденные, происходит обработка предыдущего моста, то есть текущим портом становится предыдущий по пути следования.


Структура порта в режиме моделирования по шагам

В левом верхнем углу порта отображается его номер. Далее справа (сверху вниз): текщее значение, сохраненное в памяти; значение пришедшее с пакетом; предыдуще значение памяти.
Для корневых портов добавляется подпись "root" слева внизу


Сброс и очистка

Пункт меню "Сбросить" осуществляет сброс результатов моделирования

Пункт меню "Очистить" осуществляет очистку поля программы, то есть удаление всех объектов
